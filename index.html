Features

Custom Init System

ArcirisOS uses its own init sequence built with noa-bootstrap, enabling minimal overhead and precise control over system startup and service management.

Asynchronous Package Verification

The iris package manager performs non-blocking package integrity checks during startup, ensuring system reliability without slowing down boot time.

Secure by Design

ArcirisOS tracks installed files per-package and can detect corruption or tampering in real time. All changes are verifiable, auditable, and reversible.
